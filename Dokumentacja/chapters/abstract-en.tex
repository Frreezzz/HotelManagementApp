\chapter*{Abstract in English}

The aim of the project was to create a desktop application for hotel management. The program helps a reception employee manage basic data such as guests, rooms, room types, reservations, payments, employees and additional hotel services.

The project uses C\#, WPF for the user interface and Entity Framework Core for access to the local SQLite database. The application contains a dashboard with statistics, navigation tabs, forms for entering data, tables with records and basic field validation.

The main functions of the program include adding guests, creating reservations, checking room availability, registering payments and adding or editing hotel services. The final result is a working application that implements the main assumptions of a simple hotel management system.
